Identifiers and Keywords

  • Um identificador deve seguir o padrão

    ^[letter|$|_][letter|$|_|number]*$ - {keywords}

    que o deus do regex me perdoe
  • Keywords não comuns:
    • strictfp
    • assert
    • transient
    • native
    • synchronized
    • volatile
  • CamelCase não é obrigatório. 
    class list {}

Define Class

  • Apenas uma classe public por arquivo
  • Se houver classe public, seu nome deve ser igual ao nome do arquivo

  • Ordem das declarações:

    package br.com.abevieiramota;

import java.util.List;
// package e import aplicam-se a todos os tipos
// declarados no arquivo
class Abelardo {}
// um arquivo pode ter mais de uma classe não public

  • Main válidos

    public static void main(String[] args) {}
static public void main(String[] args) {}
public static void main(String... args) {}
public static void main(String args[]) {}
public static void main(String [] args) {}
public static void main(String ... args) {}

  • Import estático

    // inválido
// static import java.lang.Integer.*;
import static java.lang.Integer.*;

  • Import com * importa apenas os tipos no nível do pacote, e não dos subpacotes

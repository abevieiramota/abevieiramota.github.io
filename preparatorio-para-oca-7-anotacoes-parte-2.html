Preparatório para OCA 7 - anotações parte 2

Published: sex 06 janeiro 2017
By Abelardo Vieira Mota

In Java.

Use Interface

  • Todos os métodos de interface são implicitamente public abstract

  • Todas variáveis de interface são implicitamente public static final

    interface Foo {
    int BAR = 32;
    void go();
}

class Zap implements Foo {
    public void go() {
        // compilation error - BAR é final
        BAR = 43;
    }
}

  • Interface não pode ter método static, final, strictfp ou native

  • Interface pode extender zero ou mais interfaces

Declare Class Members

  • Controle de acesso
    • public: toda classe com acesso à classe tem acesso ao membro
    • protected: classes do mesmo pacote da classe ou que herdem dela - observar que pode ser acessado tanto pelo próprio objeto, como com referência a objetos da classe pai; se herdado, torna-se privado para outras classes
    • default: apenas classes do mesmo pacote da classe
    • private: acessível apenas pela classe onde é declarado; não é herdado, não podendo ser sobrescrito
  • "if a subclass inherits a member, the subclass has the member"
  • A tem acesso a B.member ? = A tem acesso a B ? member é acessível a A ? : não
  • Acesso a member sem operador . -> this. implícito

  • Método final não pode ser sobrescrito

    class A {
    private final void doSomething() {};
}

class B extends A {
    public void doSomething() {
    // sem problemas, #doSomething não é herdado
    }
}

  • Se A extends B e B é abstract, então:

    • A deve implementar todos os métodos abstract de B, ou
    • A deve ser declarada abstract
  • Combinações inválidas:
    • abstract final
    • abstract static
    • abstract private

  • Var-args param deve ser o último parâmetro e só pode haver um por método

    class A {
    void doStuff(int x, int... ys) {}
    void doStuff(int x, int y) {}
    void doStuff() {
        doStuff(1, 2);
        // chama o mais específico
        // doStuff(int x, int y)
        }   
}

  • Constructor

    • mesmo nome da classe
    • todos modificadores de acesso
    • não pode static, final ou abstract
    • não tem retorno
